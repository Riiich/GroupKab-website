Algorithm Engineering


<描述> 喜歡用硬底子程式碼實現社會現象

1) 開發: 基礎程式碼層面的演算法與運作邏輯。

2) 維護: 改善系統程式碼與演算法。


<使用語言> 至少熟悉一項

1) C/C++

2) Java

3) Python

4) 或自認最熟悉的任何語言，且應試者運用的效率能夠大過上述三項


<背景需求>

1) 因為演算法與邏輯著重在科學思考與理性分析，目的是為了要把現實生活中的情境能夠資訊化呈現與模擬。 因此，程式語言只是把它實現的工具與手段。所以系統化的邏輯思考，是我們考慮的首要與必要條件。

2) 不限定任何科系或背景，只要符合第一點的邏輯思考，並且熟悉至少一種程式語言即可。

3) 願意且能夠去深入思考演算法初學者教材當中的最短路徑/排列組合...等之類的邏輯性問題。

4) 資歷需求：入門等級（剛畢業，或一到兩年經驗）

Backend Server Engineering


<描述>

1) 在既有的後端主機上，能夠維護與改善現有功能，並且，能夠為了搭配手機App而開發新功能。

2) 能夠操作/開發/管理資料庫。

3) 熟悉網路系統，開發流量負載平衡，開發分散式運算。

4) 有興趣且願意研究/分析/實測 multi-threading/multi-tasking 與 網路分散式運算 在不同使用者規模的效能評估與開發。


<使用語言> 至少熟悉一項

1) Java

2) C#/.NET

3) C/C++

4) 或自認最熟悉的任何語言，且應試者運用的效率能夠大過上述三項


<背景需求>

1) 偏好資訊相關科系，但不強制。如果有多年自學者且能夠符合職位內容皆歡迎。

2) 能夠開發後台管理者頁面為佳。

3) 資歷需求為：初階等級（至少自學玩過後端系統一年以上）

Android Application Engineering


<描述>

1) 在既有的Android APP上面，能夠維護，修改Bug，階段開發新功能。


<使用語言> 至少熟悉一項

1) Java

2) 或其他開發平台語言，但能夠兼容與熟悉既有的APP。


<背景需求>

1) 不限科系與背景，此為APP應用端的維護與開發。熟悉整個Android的開發工具，該發環境，開發流程與邏輯，即可。

2) 有任何自學或練習的作品為佳

3) 不限定一定要有Android上架經驗。

4) 資歷需求為：入門等級（剛畢業，或一到兩年經驗），初階等級（至少自學玩過Android一年以上）

iOS Application Engineering


<描述>

1) 在既有的後端主機與資料庫系統下，開發使用者端的iOS應用程式。


<使用語言> 至少熟悉一項

1) Objective-C in XCode （強烈建議）

2) 其他開發工具與語言（此項的應試者，至少要在該工具平台使用與開發兩年以上）


<背景需求>

1) 不限科系與背景，此為APP應用端的維護與開發。熟悉整個iOS的開發工具，該發環境，開發流程與邏輯，即可。

2) 有任何自學或練習的作品為佳。

3) 不限定一定要有iOS上架經驗。

4) 資歷需求為：入門等級（剛畢業，但自學玩過iOS開發一年），初階等級（至少自學玩過iOS開發有一年以上）
